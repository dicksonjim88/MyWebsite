AI-Powered COVID-19 Detection from CT Scans

This project is an artificial intelligence model designed to assist medical professionals by rapidly analyzing lung CT scans. It addresses the time-consuming challenge of manual analysis by automatically identifying potential indicators of COVID-19, providing a powerful 'second opinion' to help focus expert attention where it's needed most.

Key Features:

  • Automated Image Segmentation using a U-Net model.
  • AI-Powered Anomaly Detection to highlight regions of interest.
  • Generates an intuitive visual 'mask' for clinical review.
